#REACT Chatbot

This is a react chatbot using AWS.
It allows users to book a hotel or rent a car. 
This app was built as practice and an introduction to AWS and Lambda.

#DEv Tools Used:
`create-react-app`
<br/>
`AWS Lambda`
<br/>
`AWS Amplify`
<br/>
`AWS Lex`
